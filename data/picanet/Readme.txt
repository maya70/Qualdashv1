In admission dataset you have 194,281 admissions

pidscr	= unique identifier for patient
eventidscr = unique identifier for event (admission)
siteidscr = unique identifer for PICU

The other variables are described in the data definiitons manual.

In the activity dataset you have 636,144 days of activity. Not all admissions will have activty data as it started in 2007.

pidscr	= unique identifier for patient
eventidscr = unique identifier for event (admission)
act_no = sequential number for each day of activity 

The other variables are described in the data definiitons manual.
